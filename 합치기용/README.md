# 대한민국 월드맵 이동 프로토타입

오버쿡드2의 월드맵 이동감처럼, 사용자가 대한민국 지도 위를 부드럽게 돌아다니는 프로토타입입니다. 기본 버전은 `index.html`의 3D 브라우저 버전이고, 이전 `pygame` 파이썬 버전도 함께 남겨두었습니다. 저작권 있는 게임 에셋은 사용하지 않고 지도, 도로, 도시 노드를 코드로 직접 그립니다.

## HTML 3D 버전 실행 방법

VS Code에서 `index.html`을 열고 Live Server로 실행하면 됩니다. Live Server가 없다면 터미널에서 아래 명령을 실행한 뒤 브라우저에서 `http://localhost:8000`으로 접속하세요.

```powershell
py -m http.server 8000
```

`index.html`은 Three.js를 CDN에서 불러옵니다. 실행할 때 인터넷 연결이 필요합니다.

## HTML 조작

- `WASD` 또는 `방향키`: 이동
- `Shift`: 빠른 이동
- `Space`: 가까운 도시의 지역 팝업 열기
- `Enter`: 목포/부산/제주 근처에서 페리 이동

## Python 버전 실행 방법

```powershell
py -m pip install -r requirements.txt
py main.py
```

현재 이 PC에는 파이썬 본체가 설치되어 있지 않아 `py`가 실행할 파이썬을 찾지 못했습니다. Python 3.11 이상을 설치한 뒤 위 명령을 실행하면 됩니다.

## 조작

- `WASD` 또는 `방향키`: 이동
- `Space`: 가까운 도시 방문 처리
- `Enter`: 목포/부산/제주 근처에서 페리 이동
- `Esc`: 종료

## 구성

- 대한민국 본토와 제주/울릉도/독도를 코드 기반 도형으로 표현
- 실제 지형 모형에 가깝도록 더 촘촘한 해안선, 흰색 입체 지형, 산맥/지역 경계 표현
- 서울, 부산, 대구, 광주, 제주 등 도시 노드 표시
- 도시 사이 도로와 제주행 페리 경로 표시
- 5배로 커진 3D 지도 스케일과 다람쥐 플레이어 캐릭터
- 태백산맥, 소백산맥, 노령산맥, 차령산맥 기반의 통과 불가 산맥 장벽과 고개 지점
- 도시 팝업 상호작용과 방문 기록 버튼
- 봄 색감, 벚꽃 나무, 흩날리는 꽃잎 표현
- 카메라 추적, 미니맵, 방문 완료 UI 포함
